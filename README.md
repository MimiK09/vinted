# API - Vinted

## User

### /user/signup (POST)

Create a new user

### /user/login (POST)

Log a user

## Offer

### /offers/ (GET)

List of offers.

### /offer/:id (GET)

Get an offer

### /offer/publish (POST)

Create a new offer

### /offer/update/:id (PUT)

Update an offer

### /offer/delete/:id (DELETE)

Delete an offer
